Regras de conversão

PNG / JPG / WebP→AVIF / WebP
GIF animado→WebP / WebM
Animação grande→WebM
+ miniatura 320px+ dedup por hash

Re-encode forçado (até webp grande encolhe) → AVIF quando suportado. Miniatura p/ feeds. Conteúdo igual = guardado 1x. GIF→WebM animado é server-side.

Limpeza ao banir

Remove mídia pessoal, mantém registro mínimo legal. Modo dryRun por padrão.

Removido
  • 🖼️ Avatar
  • 🏞️ Banner do perfil
  • 📷 Imagens do perfil
  • 🗂️ Uploads temporários
  • ♻️ Versões em cache
Mantido
  • 🆔 ID do usuário
  • 📝 Motivo do ban
  • 📅 Data do ban
  • 🛡️ Logs de moderação
  • 👤 Registro mínimo

Conteúdo público (posts/comentários) não é removido sem regra de moderação. Imagens removidas viram placeholder. Ao voltar, o usuário reenviar avatar/banner manualmente.
